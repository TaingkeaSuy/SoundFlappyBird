Flappy Bird is a simple 2D side-scrolling arcade game where the player controls a small bird trying to fly between sets of pipes without hitting them.

+Gameplay Overview

The player controls a bird that:

Automatically moves forward (to the right).

Falls downward due to gravity.

Jumps upward when the player presses a key (e.g., Spacebar) or makes a sound (voice input).

+The objective is to:

Pass through gaps between pipes.

Avoid colliding with pipes or the ground.

Survive as long as possible to achieve a high score.
